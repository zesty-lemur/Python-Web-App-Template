# Web App Template

Template repo for secure web apps

## Usage

1. Clone the repo:

```bash
git clone https://github.com/zesty-lemur/Web-App-Template
cd https://github.com/zesty-lemur/Web-App-Template <app-name>
```

2. Run the setup script:

```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```

```powershell
.\scripts\setup.ps1
```
